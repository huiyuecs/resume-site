# Resume Website

个人简历网站 / Personal Resume Website

## 简介

这是一个简单的GitHub Pages网站，用于展示我的简历。

This is a simple GitHub Pages website to showcase my resume.

## 文件说明

- `index.html` - 网站主页
- `elena_zhou_github.pdf` - 简历PDF文件

## 如何部署到GitHub Pages

1. 将本仓库推送到GitHub
2. 在仓库设置中，进入 "Pages" 部分
3. 选择 "Deploy from a branch"
4. 选择 `main` (或 `master`) 分支
5. 选择 `/ (root)` 作为源文件夹
6. 点击保存

网站将在几分钟内部署到：`https://[你的用户名].github.io/my-resume/`

## 使用说明

- 在浏览器中打开 `index.html` 可以本地预览
- PDF文件会嵌入在网页中，也可以点击下载按钮下载
- 网站支持响应式设计，在手机和电脑上都能良好显示

## 技术栈

- HTML5
- CSS3
- GitHub Pages
